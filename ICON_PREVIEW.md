# Icon Preview 说明（插件生成页面概览）

本文档简要记录生成的 icon 预览页交互、控制项以及开发时的构建方式，便于查阅和联调。

## 页面交互
- 图标列表点击即可在下方预览区展示详情，不再有悬停复制。
- 预览区提供三种复制按钮：复制图标（组件片段）、复制路径、复制样式（包含当前颜色与旋转样式）。
- 颜色面板：内置拾色器 + 文本输入，支持 Hex / RGB 两种格式一键切换并保持同步。
- 旋转控制：左右各有 “-10” / “+10” 快捷按钮，输入框居中并隐藏原生上下箭头，步进 10°。

## 样式细节
- 网格卡片只在悬停时高亮，名称在卡片内展示，去掉了 tooltip hover 复制。
- 预览区撑满容器宽度，页面禁用横向滚动，整体背景/边框颜色与当前主题保持一致。

## 开发与构建
- 预览页文件由插件自动生成（`src/generator/templates.ts` 中定义模板）。
- 普通构建：`pnpm build`（或 `npm run build`/`yarn build`）。
- 开启监听：`pnpm build:watch`（或对应 npm/yarn 命令），同时 watch ESM/CJS 打包与 `tsc -p tsconfig.build.json --watch`，便于实时预览调整。
